# CLI Tool

The `create-agentic-zero` CLI lets you scaffold a fully configured Agentic Zero project in seconds.

## Usage

```bash
npx create-agentic-zero my-app
```

This will:

1. **Prompt** you for a project name (if not provided) and template
2. **Scaffold** the project from a template
3. **Install** all dependencies
4. **Initialize** a git repository

## Options

```bash
create-agentic-zero [project-directory] [options]
```

| Option | Description |
|:--|:--|
| `[project-directory]` | Directory name for the new project |
| `-t, --template <template>` | Template to use (`nextjs` or `vite`) |

## Non-Interactive Mode

You can skip the interactive prompts by providing all arguments:

```bash
npx create-agentic-zero my-app --template nextjs
```
