# API Reference

## AgenticProvider

The root component that initializes the agent.

```tsx
<AgenticProvider config={{ 
  autoStart: boolean, 
  debug: boolean,
  bridge: BridgeConfig 
}}>
  {children}
</AgenticProvider>
```

| Prop | Type | Description |
| :--- | :--- | :--- |
| `config.autoStart` | `boolean` | If true, starts the journey automatically on mount. |
| `config.debug` | `boolean` | Enables the **Agentic Console** overlay. |

## useAgent

Hook to access the agent state.

```tsx
const { state, send, bridge } = useAgent();
```

- **state**: The current XState state object.
- **send**: Function to send events to the machine (e.g., `send({ type: 'USER_INTERRUPT' })`).
- **bridge**: Direct access to the `BridgeOrchestrator` instance.
