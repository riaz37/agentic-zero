# Agentic Zero

**Transform any website into an autonomous concierge experience.**

## Architecture

The framework is built on a **Bridge Pattern**:

1. **The Brain (XState)** â€” Decides *intent* (e.g., "Go to Pricing")
2. **The Orchestrator** â€” Translates intent into *action* (e.g., "Scroll to #pricing")
3. **The Bridge** â€” Executes action and *verifies* result (e.g., "Is #pricing visible?")
